## 项目详情模板

### 详情格式

```markdown
**【项目名称】项目描述**
- 一句话定位：{一句话描述这个项目是什么}
- 核心技术栈：{项目使用的主要技术}
```

### 一句话定位示例

| 项目类型 | 一句话定位示例 |
|---------|---------------|
| 框架/库 | 主流CTR模型的统一实现，包括DeepFM、DIN、DIEN等 |
| 工具 | 基于Transformer架构的序列推荐模型 |
| 系统 | 全链路推荐系统，覆盖召回、排序、重排各阶段 |
| 教程 | 推荐系统入门教程，包含理论讲解和代码实践 |

### 核心技术栈示例

- 简单项目：`Python`、`PyTorch`
- 中等项目：`PyTorch`、`Transformers`、`Hugging Face`
- 复杂项目：`TensorFlow Serving`、`Kubernetes`、`Redis`、`Faiss`

## 输出检查清单

在输出报告前，确认以下项：

- [ ] 热门项目表格包含3个项目
- [ ] 新项目表格包含3个项目
- [ ] 每个项目有GitHub链接
- [ ] Star数格式正确（热门用k、新项目用原始数字）
- [ ] 创建时间格式为具体日期（YYYY-MM-DD）
- [ ] 学习坡度用星标表示（⭐）
- [ ] 潜力指数用星标表示（⭐⭐⭐⭐⭐）
- [ ] 每条推荐理由包含"如需进一步学习，建议调用codebase-to-course技能生成代码项目交互式教程"
- [ ] 每个项目有"一句话定位"
- [ ] 每个项目有"核心技术栈"
- [ ] 数据来自真实搜索结果

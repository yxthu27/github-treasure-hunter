# github-treasure-hunter
# 拓码哥子 (GitHub Treasure Hunter) - 帮助转码人士根据岗位需求精准挖掘GitHub上的宝藏项目

> 像养育电子宠物一样培育你的代码能力！

## 简介

**拓码哥子**（谐音"拓麻歌子" Tamagotchi）是一个帮助转码学习者根据岗位技能需求，精准挖掘 GitHub 宝藏开源项目的 AI Skill。

它通过**两步闭环**（场景定位 + 项目推荐），既有热门成熟项目的稳重内涵，又有新型项目的最新灵感，助你早日成为码届之王！

## 核心功能

### 1. 场景卡片确认
将模糊的"我想学..."转化为明确的技能场景：
- 分析用户输入，识别技术方向
- 匹配或生成 2-4 个场景卡片
- 帮助用户精准定位学习目标

### 2. 项目推荐报告
输出结构化的项目推荐：
- **热门项目**（3个）：Star > 1000，社区验证的优质项目
- **新项目淘金**（3个）：Star < 500，创建时间在用户指定范围内，有潜力的新项目

## 使用方法

### 触发方式
对 AI 说出以下任意一种表达：
- "我想学推荐系统"
- "我想转行到前端"
- "推荐一些 AI 相关的项目"
- "帮我找一些适合入门的开源项目"

### 对话流程

**第一步：场景定位**
```
用户：我想学搜广推
AI："搜广推"是一个很大的方向，让我帮你定位：

【场景卡片A】电商推荐系统
- 核心任务：召回→排序→重排
- 典型项目：用户行为建模、多目标优化
- 技术栈：Faiss/Milvus, XGBoost, TensorFlow Serving

【场景卡片B】搜索引擎...

你想查看多久之内创建的新项目？（1个月/3个月/6个月）
```

**第二步：项目推荐报告**
```
## 项目推荐报告

### 热门项目推荐（3个）
| 项目名称 | Star | 最近更新 | 学习坡度 | 推荐理由 |
|---------|------|---------|---------|---------|
| [DeepCTR](...) | 8.2k | 1周前 | ⭐⭐⭐ | 主流CTR模型集合... |

### 新项目淘金（3个）
| 项目名称 | 创建时间 | Star | 潜力指数 | 推荐理由 |
|---------|---------|------|---------|---------|
| [RecFormer](...) | 2026年01月15日 | 127 | ⭐⭐⭐⭐ | Transformer推荐模型... |
```

## 适用人群

- 🎯 **转码学习者**：根据岗位需求精准匹配学习资源
- 🚀 **技术转型者**：快速了解新领域的高质量项目
- 🔍 **开源探索者**：发现热门项目的同时挖掘潜力新项目

## 技术特点

- **纯自然语言实现**：无需脚本，完全通过 AI 能力实现
- **灵活可扩展**：场景卡片和领域分类均可动态扩展
- **数据真实**：所有项目信息来自实时 GitHub 搜索
